Judges' comments:

Try:

ph 40 < ph.c > foo.c; cc foo.c -o ph
ph 20 < a_C_prog.c > bar.c; cc bar.c

The judges used this program to fold up very long lines which the BSD fold(1) was unable to process. This program has been added to the contest tool chest.

On System V systems, compile with: -Dindex=strchr To compile on a 16 bit machine, change 300000’s to 30000.
